## 🎯 为什么中文市场是蓝海

```
全球 Agent Skills 生态         中文 Agent Skills 生态
```

## 🚀 如何贡献

1. **提交技能**：Fork 本仓库，在 `skills/` 下创建你的技能目录，包含 `SKILL.md`
2. **适配中文平台**：把现有的英文技能改成中文版（API、平台、语言）
3. **创建中国独有技能**：小红书/抖音/微信/闲鱼/施工方案等场景
